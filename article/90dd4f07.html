标签、控件

view

可以将东西输出出来，是最基本最常用的标签

image

image标签用来链接图片，但是这里要对其属性做说明

为了更好的说明，我们通过一张图片来进行直观的感觉
原图：

当我们不使用任何的属性约束，单纯的使用image标签引入时


可以从预览器中看到，图片比例很怪。这是因为在微信小程序开发中，image组件默认的宽高为320 px * 240 px,所以我们只能通过属性来更改图片显示。

mode-scaleToFill

scaleToFill
缩放模式，不保持纵横比缩放图片，使图片的宽高完全拉伸至填满 image 元素

拉伸图片至填满组件

mode-aspectFit

缩放模式，保持纵横比缩放图片，使图片的长边能完全显示出来。也就是说，可以完整地将图片显示出来。


我们可以看到，图片被完整的显示出来了，但是左边却空出来了一部分，这是因为该属性以长边(宽)为先，保持原图比例，导致该图片为240px * 240px，没有填满左边的组件。

mode-aspectFill

缩放模式，保持纵横比缩放图片，只保证图片的短边能完全显示出来。也就是说，图片通常只在水平或垂直方向是完整的，另一个方向将会发生截取。
从图上结果我们可以理解为原图放大之后被下边界进行了裁剪

mode-widthFix

缩放模式，宽度不变，高度自动变化，保持原图宽高比不变
这是最常用的属性

mode-left、top等

裁剪模式，不缩放图片，只显示图片的对应区域

mode-top left等

裁剪模式，不缩放图片，只显示图片的左上边（对应斜方向）区域

swiper

swiper组件通常用来构建轮播图，通过内置组件来实现轮播图效果

swiper的常用属性如下

swiper-autoplay

开启之后会实现动态轮播,同时也会激活interval，默认值为5000

swiper-interval

页面之间进行切换的时间间隔，默认单位为ms，默认值为5000ms，也就是5秒钟。

swiper-circular

开启之后，当播放到最后一张图片时，会继续往下从第一张循环，而不是每次到了最后一张然后又倒着快速回到第一张开始。

注意

  • 如果在 bindchange 的事件回调函数中使用 setData 改变 current 值，则有可能导致 setData 被不停地调用，因而通常情况下请在改变 current 值前检测 source 字段来判断是否是由于用户触摸引起。
  • swiper组件存在默认样式，宽度为100%，高度为150px,但是图片也存在宽度和高度。但是swiper标签无法由内容撑开，所以我们链接的图片要进行换算来保证它的正常显示。换算方法：
    swiper宽度 / swiper高度 = 原图宽度 / 原图高度
    所以我们只需要计算出swiper的高度就可，可以使用快捷计算函数calc

navigator

导航组件，类似于html中的a标签

但是需要对其属性值进行设置，不加属性下，无法跳转至tabber页面，具体属性如下

navigator-target

定义跳转目标，可选值为self和miniprogram，默认值为self
当target为self，指的是跳转到小程序内部的某地
当target为miniprogram时，跳转到其他小程序

navigator-url

当前小程序内部跳转的链接，就是pages页面中的那个路径，注意修改../

navigator open-type

跳转方式，具体如下

navigate 保留当前页面，跳转到应用内的某个页面，但是不能跳转到tabbar页面
redirect 关闭当前页面，跳转到应用内的某个页面，但是不能跳转到tabbar页面
switchTab 跳转到tabbar页面，并关闭其他所有非tabbar页面
reLaunch 关闭所有页面，打开到应用内的某个页面
navigateBack 关闭当前页面，返回上一级或多级，默认值为1
exit 退出跳转到的小程序

保留页面和关闭页面，就是在左上角有没有返回按键

保留页面

关闭页面

button

按钮标签，可以在页面中提供一个按钮

button-type

当type值为空，为默认样式，
当type值为primary，为绿色样式，
当type值为warn，为红色样式。

button-plain

按钮是否镂空、按键是否透明

button-loading

前置加载图标

icon

图标。组件属性的长度单位默认为px，2.4.0起支持传入单位(rpx/px)。

不同图标对应了type属性的值，分别为success, success_no_circle, info, warn, waiting, cancel, download, search, clear

radio/checkbox

单选框/复选框

基本的标签、控件到这里就结束了，自定义控件的开发会再开一片新的博客。
